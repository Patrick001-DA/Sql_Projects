Loan Analysis SQL Project

Project objective
Practice SQL analysis on loan applicant data and answer business-style questions about income, defaults, education, credit score, and loan purpose.

Table used
- loan_data

Skills practiced
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- CASE
- Aggregation functions like COUNT, AVG
- Filtering with BETWEEN and AND

Suggested workflow
1. Run schema.sql
2. Run insert_data.sql
3. Run queries.sql one query at a time
4. Add more questions such as:
   - highest loan by education level
   - average credit score by gender
   - applicants with loan_percent_income above 0.40
   - defaulted applicants with low credit score
