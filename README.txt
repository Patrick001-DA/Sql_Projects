Banking Database SQL Project

Project objective
Build a small banking database and practice real SQL skills using six related tables:
- customers
- accounts
- branches
- loans
- employees
- transactions

Skills practiced
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- CONCAT
- Subqueries

Suggested workflow
1. Open OneCompiler or DB Fiddle.
2. Run schema.sql first.
3. Run insert_data.sql next.
4. Run the queries from queries.sql one by one.
5. Add your own questions and answers as you learn more.
